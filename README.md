# 🚗 Dream Car · Чайтанья

Бесконечная 3D-гонка на вымышленной машине с динамическим временем суток, фарами, выхлопом и музыкой.

## 🌟 Особенности

- **Бесконечная дорога** — полоски разметки и ландшафт зациклены.
- **Смена дня и ночи** — ползунок времени (0–24 ч) плавно меняет освещение и цвет неба.
- **Фары** — кнопка включает/выключает ближний свет и прожектор.
- **Выхлоп** — кнопка запускает облачко частиц из выхлопной трубы.
- **Музыка** — 16 встроенных треков SoundHelix + возможность добавить свою MP3-ссылку.
- **Управление камерой** — можно крутить вид (OrbitControls) мышкой или пальцем на телефоне.

## 🎮 Управление

| Кнопка              | Действие                          |
|---------------------|-----------------------------------|
| Ползунок времени    | меняет день/ночь                  |
| «Фары Выкл/Вкл»     | включает/выключает свет фар       |
| «Выхлоп Выкл/Вкл»   | включает/выключает дым из трубы   |
| Выбор трека + «Вкл» | запускает выбранную музыку        |
| Поле ввода + «Добавить» | добавляет свою MP3-ссылку в список |

## 🖥️ Технологии

- Three.js (3D-графика)
- OrbitControls (камера)
- HTML/CSS (интерфейс)
- JavaScript (логика)
