# 🚗 Dream Car

> Infinite 3D driving simulator with day/night cycle, headlights, exhaust, and built‑in music player.

---

## 🌟 Features

- **Infinite road** — road markings and landscape loop seamlessly, creating the illusion of endless driving.
- **Day/Night cycle** — a slider (0–24h) smoothly changes lighting, sky color, and sun/moon intensity.
- **Headlights** — toggleable front lights (spot + point light) for better visibility in the dark.
- **Exhaust** — animated particle cloud from the exhaust pipe makes the car feel alive.
- **Music player** — 16 built‑in SoundHelix tracks (Creative Commons) + ability to add your own MP3 links.
- **Camera control** — rotate and zoom with mouse/touch (OrbitControls).

---

## 🎮 Controls

| Element                  | Action                                      |
|--------------------------|---------------------------------------------|
| **Time slider**          | Change time of day (0:00 – 24:00)           |
| **Headlights button**    | Toggle front lights on/off                  |
| **Exhaust button**       | Toggle exhaust particles on/off             |
| **Track dropdown + Play**| Select a track and start/stop music         |
| **Input field + Add**    | Add a direct MP3 link to the playlist       |

---

## 🖥️ Technologies

- **Three.js** — 3D rendering engine
- **OrbitControls** — interactive camera
- **HTML/CSS** — responsive UI with dark glass‑morphism style
- **JavaScript (ES modules)** — all logic in a single file

# 🚗 Dream Car (Русская версия)

> Бесконечный 3D-симулятор вождения с динамической сменой дня и ночи, фарами, выхлопом и встроенным музыкальным плеером.

---

## 🌟 Особенности

- **Бесконечная дорога** — полоски разметки и ландшафт зациклены, создавая иллюзию непрерывного движения.
- **Смена дня и ночи** — ползунок (0–24 ч) плавно меняет освещение, цвет неба и интенсивность солнца/луны.
- **Фары** — кнопка включает/выключает передний свет (прожектор и точечный источник) для лучшей видимости в темноте.
- **Выхлоп** — анимированное облачко частиц из выхлопной трубы делает машину более живой.
- **Музыка** — 16 встроенных треков SoundHelix (Creative Commons) + возможность добавить свою MP3-ссылку через поле ввода.
- **Управление камерой** — можно крутить и приближать вид мышью или пальцем (OrbitControls).

---

## 🎮 Управление

| Элемент                  | Действие                                      |
|--------------------------|-----------------------------------------------|
| **Ползунок времени**     | Изменяет время суток от 0:00 до 24:00         |
| **Кнопка «Фары»**        | Включает/выключает передние фары              |
| **Кнопка «Выхлоп»**      | Включает/выключает дым из выхлопной трубы     |
| **Выпадающий список + кнопка «Музыка»** | Выбор трека и запуск/остановка музыки |
| **Поле ввода + кнопка «Добавить»** | Добавляет прямую ссылку на MP3 в список треков |

---
